# 마녀봇 개발 로드맵

마스토돈 커뮤니티 관리 봇의 개발 계획 및 진행 상황입니다.

---

## ✅ 완료된 기능 (v1.0)

### 핵심 시스템
- [x] 재화 지급 시스템 (답글 기반 자동 정산)
- [x] 활동량 모니터링 (4가지 위험 유형)
  - 고립 위험 (접속 인원의 절반 미만과 대화)
  - 편향 위험 (특정인과 10% 이상 대화)
  - 회피 패턴 (주요 멤버 회피)
  - 답글 미달 (48시간 내 5개 미만)
- [x] 경고 시스템 (수동 발송)
- [x] 휴식 관리 (셀프 서비스)
- [x] 일정 관리 (이벤트, 전역 휴식기간)
- [x] 상점 시스템 (아이템 구매/양도)

### 관리자 웹 인터페이스
- [x] OAuth 로그인 (마스토돈 계정 연동)
- [x] 대시보드 (실시간 통계)
- [x] 유저 관리 (활동량 모니터링, 휴식 현황)
- [x] 재화 관리 (개별 유저 지갑, 거래 내역)
- [x] 상점 관리 (아이템 CRUD, 호그와트 테마)
- [x] 일정 관리 (이벤트 등록, 리뉴얼 기간 설정)
- [x] 스토리 예약 관리 (이벤트 기반 포스트 시스템)
- [x] 공지 예약 관리
- [x] 시스템 설정 (시간 선택기)
- [x] 관리 로그 (9가지 액션 타입, 날짜 필터, CSV 다운로드)

### UI/UX 개선
- [x] SVG 목업 디자인 (Bootstrap 5 색상 체계)
- [x] 시간 선택기 (체크/정산 주기)
- [x] 날짜 범위 필터 (로그 조회)
- [x] 모달 UI (아이템 추가, 이벤트 등록)
- [x] 개별 유저 지갑 UI (거래 내역 디버깅)

### 문서화
- [x] 관리자 가이드 (웹 UI 사용법, 유지보수 포함)
- [x] 배포 가이드 (서버 설정 + Docker 통합)
- [x] 시스템 아키텍처
- [x] 데이터베이스 스키마
- [x] API 설계 문서
- [x] 긴급 대응 절차

### 인프라
- [x] Docker Compose 구성
- [x] 멀티 스테이지 Dockerfile
- [x] Redis 캐싱
- [x] Celery 백그라운드 작업
- [x] 자동 백업 시스템
