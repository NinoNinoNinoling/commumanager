# Part 5: 관리자 웹 UX 설계

---

## 1. 설계 원칙

### 1.1 비개발자 중심 설계

**대상 사용자**:
- 일반 관리자: 웹 기본만 아는 수준
- 기술 관리자: 우분투, 웹 운용 가능

**핵심 원칙**:
1. ❌ 전문용어 NO - "DB 쿼리" 대신 "목록 조회"
2. 👀 한눈에 파악 - 중요한 정보는 첫 화면에
3. 🛡️ 실수 방지 - 위험한 동작엔 확인 팝업
4. ✅ 명확한 피드백 - "완료" / "실패" 즉시 표시
5. 📝 로그 남기기 - 누가 언제 뭘 했는지 기록

---

## 2. 전체 메뉴 구조

```
관리자 대시보드
├── 🏠 홈 (대시보드)
│   ├── 오늘의 알림
│   ├── 전체 통계
│   ├── 전체 유저 활동 현황 (48시간 기준)
│   └── 봇 상태
│
├── ⚠️ 활동량 관리
│   ├── 자동 경고 내역
│   ├── 수동 경고 보내기
│   └── 휴식 관리
│
├── 💰 재화 관리
│   ├── 전체 유저 목록
│   └── 재화 지급/차감
│
├── 🏪 상점 관리
│   ├── 아이템 목록
│   ├── 아이템 추가
│   └── 아이템 수정/삭제
│
├── 🛠️ 시스템 설정
│   ├── 활동량 체크 기준
│   ├── 재화 지급 설정

│
└── 📝 관리 로그
    └── 전체 작업 기록
```
