# 프로젝트 개요

## 목표
휘핑 에디션 마스토돈에 활동량 기반 자동 운영 관리 시스템 추가

- 휘핑 에디션: https://github.com/whippyshou/mastodon

## 핵심 기능 우선순위

### 1순위: 활동량 체크 및 경고
- 48시간 내 답글 N개 미달 시 자동 경고
- 하루 2회 체크 (오전 4시, 오후 4시)
- 휴식계: 유저/관리자가 휴식 등록 → 체크 제외
- 역할 필터: user만 체크, admin/system/story 제외
- 관리자 봇으로 비공개 툿 발송

### 2+3순위: 재화 지급 + 상점
- 답글 작성 시 실시간 재화 지급 (중복 방지)
- 커스텀 지급 방식: N개당 M원 (DB 설정)
- 상점: 수집용 아이템 무한 구매
- 봇 명령어: `@봇 내재화`, `@봇 상점`, `@봇 구매 [아이템]`

### 4순위: 일반 유저용 웹 (추후)
- 타 유저 프로필 조회
- 재화 랭킹
- 아이템 소유 현황

### 6순위: 게임 시스템 (추후)
