# 마녀봇 기능 목록

## 시스템계정 명령어 (@봇)

**역할**: 유저 친화적 시스템 봇
**응답**: 모든 응답 DM으로 전송

### 재화 관리
- `@봇 내재화`: 보유 재화 조회 (현재/누적 획득/사용)

### 상점
- `@봇 상점`: 구매 가능 아이템 목록 (카테고리별, 가격 표시)
- `@봇 구매 [아이템명]`: 아이템 구매 (재화 차감, 부족 시 실패)
- `@봇 내아이템`: 보유 아이템 목록 (획득일 포함)

### 휴식 관리
- `@봇 휴식 N`: 오늘부터 N일간 휴식 등록 (자동 승인)
  - 예: `@봇 휴식 2` → 오늘, 내일 (2일)
  - 관리자 웹에서 💤 표시
- `@봇 휴식 해제`: 현재 휴식 즉시 종료

### 일정 조회
- `@봇 일정`: 다음 전역 휴식 전까지 일정 조회 (전역 휴식 없으면 +30일)
- `@봇 일정 이번달`: 이번 달 전체
- `@봇 일정 다음주`: 7~14일 이내

### 도움말
- `@봇 도움말` 또는 `@봇 help`: 명령어 목록

**응답 예시:**
```
🤖 봇 명령어 도움말

[재화 및 아이템]
• @봇 내재화 - 보유 재화 조회
• @봇 상점 - 아이템 목록
• @봇 구매 [아이템명] - 아이템 구매
• @봇 내아이템 - 보유 아이템 조회

[휴식 관리]
• @봇 휴식 N - N일간 휴식 등록
• @봇 휴식 해제 - 휴식 해제

[정보 조회]
• @봇 일정 - 이번 주 일정
• @봇 공지 - 최신 공지 및 안내
• @봇 도움말 - 이 도움말 표시
```

### 공지 조회
- `@봇 공지`: 공개 설정된 최신 공지 3개 + 다가오는 일정(7일) + 진행 중 이벤트

**응답 예시:**
```
📢 공지 및 안내

[최신 공지]
• 2025-11-20: 연말 이벤트 안내
• 2025-11-18: 활동량 기준 변경 안내
• 2025-11-15: 신규 아이템 출시

[다가오는 일정] (7일 이내)
• 11/30 (토) 🎉 커뮤니티 모임
• 12/03 (화) 📢 중요 공지 예정
• 12/25 (수) 🏖️ 크리스마스 휴식기간 (출석 트윗 없음)

[현재 진행 중]
• 출석 연속 30일 달성 이벤트

자세한 내용은 @봇 일정 또는 관리자 웹에서 확인하세요.
```

**참고**: scheduled_posts 테이블의 is_public=1인 공지만 표시

## 자동 시스템

### 시스템계정 (@봇)

**재화 지급 (일괄 정산)**
- 하루 2회 (오전 4시, 오후 4시)
- 이전 정산 이후 답글 수 집계
- 지급 공식: `(답글 수 / N) * M원` (예: 100개당 10원)
- 중복 방지: transactions에 기록된 답글 제외

**출석 체크**
- 매일 오전 10시 출석 트윗 자동 발행
- 출석 가능 시간: 오전 10시 ~ 자정 (24:00)
- 유저가 답글 시 재화 지급 (50원 고정)
- 중복 방지: UNIQUE(user_id, attendance_post_id)
- 휴식 중 유저도 출석 가능
- 전역 휴식기간: 출석 트윗 발행 자체를 막음
- 출석 트윗에 최근 정산 완료 시각 표시
- 출석 트윗 멘트: 관리자 웹에서 지정 가능 (settings.attendance_tweet_template)

**출석 트윗 예시:**
```
🌟 오늘의 출석 체크!
이 트윗에 답글 달아주세요!

💰 최근 정산 완료: 오후 4시
재화 지급이 완료되었습니다.
```

### 총괄계정

**유저 등록**
- 총괄계정 팔로우 시 즉시 DB 등록 (follow 이벤트)
- Redis 캐시 갱신 (1시간 TTL)
- 백업: 답글 재화 지급 시 Lazy Creation

**공지 발행**
- scheduled_posts의 announcement 타입 발행
- 관리자 웹에서 예약

### 스토리계정

**스토리 발행**
- scheduled_posts의 story 타입 발행
- 관리자 웹에서 예약

### 감독봇

**소셜 분석 (매일 새벽 4시)**
- user_stats에 분석 결과 저장
- 자동 경고 없음 (관리자 수동 판단)
- 분석 항목:
  - 편중: 특정인과 대화 30% 초과
  - 고립: 대화 상대 7명 미만
  - 비활동: 7일 접속률 50% 미만

**경고 발송 (수동)**
- 관리자가 판단 후 발송
- 활동량 미달 경고
- 소셜 분석 경고

**운영진 알림**
- admin_notice 타입 발행 (private, 운영진 전용)
- 크리티컬 에러 알림 (DB 실패, API 오류 등)

## 관리자 웹 기능

### OAuth 로그인
**접근 권한**:
1. 총괄계정 OAuth 로그인 (모든 권한)
2. role='admin' 유저 OAuth 로그인 (역할 변경 제외)

**권한 확인**:
- 로그인 시 users.role 확인
- role='admin' 또는 총괄계정이면 접근 허용
- 세션 기반 인증 (24시간)

### 대시보드
- 전체 통계 (유저 수, 총 재화, 48h 답글)
- 오늘의 알림 (경고 유저, 휴식 중)

### 유저 관리
- 유저 목록 (검색, 필터, 정렬)
- 상태 표시: 🟢 정상 / 🔴 기준 미달 / 💤 휴식 중 / ⚠️ 경고 누적
- 개인별 페이지:
  - 기본 정보 (재화, 재화 이력, 아이템)
  - 활동량 조회 (실시간)
  - 소셜 통계 (대화 상대, 접속 패턴)
  - 경고 이력

### 활동량 관리
- 개인별 활동량 조회 (48시간 답글 수, PostgreSQL 실시간)
- 기준 충족 표시: 🟢 정상 / 🔴 미달 / 💤 휴식 중
- 경고 이력 조회 (필터: 1회/2회/3회 이상)
- 수동 경고 발송 (메시지 작성, DM 발송)
- 휴식 관리 (목록, 등록, 해제)

**활동량 분석 페이지** (삼성 헬스 스타일):
- **오늘 분석**: 현재까지 답글 수, 기준 대비 진행률, 시간대별 활동
- **매일 분석**: 최근 7일/30일 일별 답글 수 추이 (차트)
- **종합 분석**: 평균 답글 수, 최고/최저 기록, 패턴 분석
- **과거 비교**: 오늘 vs 어제, 이번 주 vs 지난 주

**참고:**
- 자동 경고 없음
- 관리자가 수동으로 발송
- 활동량은 실시간 조회만 (DB 저장 안 함)

### 재화 관리
- 유저별 재화 현황 (정렬: 많은 순/적은 순)
- 재화 조정 (지급/차감, 사유 필수, admin_logs 기록)
- 거래 내역 조회 (필터: 유저별/기간별/타입별)

### 아이템 관리
- 아이템 등록 (이름, 설명, 가격, 카테고리, 재고)
- 아이템 수정/삭제
- 판매 중단/재개

### 일정/이벤트 관리
- 일반 이벤트 등록 (커뮤니티 모임, 행사, 기념일)
  - 단일: end_date = NULL
  - 기간: end_date != NULL
  - 타입: event/holiday/notice
- 전역 휴식기간 설정 (체크박스, 출석 트윗 발행 비활성화)
- 목록 보기 (월별 캘린더/주별 리스트)
- 필터 (타입별, 전역 휴식기간만)
- 수정/삭제

### 역할 관리
**권한**: 총괄계정만 가능
- 관리자 추가 (role 변경: user → admin)
- 관리자 제거 (role 변경: admin → user)
- 관리자 목록 조회

### 시스템 설정

**활동량 조회 설정**
- 조회 기간 (시간) (기본: 48)
- 최소 답글 수 (기본: 20)

**재화 지급 설정**
- 기준 답글 수 (N개) (기본: 100)
- N개당 지급 재화 (M원) (기본: 10)

**출석 체크 설정**
- 출석 트윗 발행 시간 (기본: 10:00)
- 출석 트윗 템플릿 (기본: "🌟 오늘의 출석 체크!\n이 트윗에 답글 달아주세요!")
- 기본 출석 보상 (기본: 50)
- 시스템 활성화/비활성화

**휴식 관리 설정**
- 최대 휴식 기간 (일) (기본: 90)
- 봇 셀프 등록 허용/불허

**소셜 분석 설정**
- 고립 판정 기준 (N명 미만) (기본: 7)
- 편중 판정 기준 (비율) (기본: 0.3)
- 비활동 판정 기준 (접속률) (기본: 0.5)

**경고 및 아카이빙**
- 툿 아카이빙 경고 임계값 (기본: 3)
- 어드민 마스토돈 계정 ID

**타임존**
- Asia/Seoul

### 커뮤니티 건강도 (소셜 분석)
- 자동 분석 (매일 새벽 4시, user_stats 저장, 자동 경고 없음)
- 대시보드:
  - 🔴 대화 편중 (특정인과 30% 초과)
  - 🟡 고립 위험 (대화 상대 7명 미만)
  - 🟠 비활동 (접속률 50% 미만)
  - 🟢 건강한 유저
- 필터: 편중/고립/비활동/전체 문제 유저
- 유저 상세: 대화 분석(48h), 접속 패턴(7일), 문제 플래그, 경고 이력
- 수동 경고 발송: 관리자가 회의/판단 후 발송

### 경고 관리
- 경고 목록 (유저별, 날짜별)
- 경고 템플릿 관리
- 수동 경고 발송 (유저 선택, 메시지 작성)

### 아카이빙 (경고 3회 누적 시)
- 아카이빙 후보 목록 (경고 3회 이상)
- 아카이빙 실행 (툿 다운로드, DB 기록, 밴)
- 아카이빙 이력 조회

### 관리자 로그
- 모든 관리자 액션 기록 (재화 조정, 경고 발송, 설정 변경 등)
- 필터 (날짜, 관리자, 액션 타입)
