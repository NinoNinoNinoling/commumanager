# 마녀봇 기능 목록

## 봇 명령어 (유저용)

### 재화 관리
```
@봇 내재화
```
- 현재 보유 재화 확인
- 누적 획득/사용 재화 확인
- DM으로 응답

### 상점
```
@봇 상점
```
- 구매 가능한 아이템 목록
- 카테고리별 분류
- 가격 표시

```
@봇 구매 [아이템명]
```
- 아이템 구매
- 재화 차감
- inventory에 추가
- 재화 부족 시 실패

```
@봇 내아이템
```
- 보유 아이템 목록
- 획득 날짜 표시

### 휴식 관리
```
@봇 휴식 [날짜]까지
```
- 활동량 체크 제외 등록
- 예: `@봇 휴식 2024-11-25까지`
- 기간 동안 경고 발송 안 됨

### 게임 (추후 구현)
- 봇 대화형 게임 시스템
- 재화를 걸고 승부
- 구체적인 게임 종류는 추후 결정

### 일정 조회
```
@봇 일정
```
- 이번 주 커뮤니티 일정 조회 (오늘 ~ 7일 이내)
- 다가오는 이벤트 표시
- 전역 휴식기간 안내
- 공휴일/특별 이벤트 표시

```
@봇 일정 이번달
```
- 이번 달 전체 일정 조회

```
@봇 일정 다음주
```
- 다음 주 일정 조회 (7일 ~ 14일 이내)

### 도움말
```
@봇 도움말
```
- 사용 가능한 명령어 목록
- 간단한 사용법

## 자동 시스템

### 재화 지급 (24시간)
- 답글 작성 시 자동 지급
- 커스텀 설정 (N개당 M원)
- 중복 지급 방지 (status_id)
- 모든 거래 DB 기록

### 출석 체크 (하루 1회, Asia/Seoul 타임존 기준)
- **자동 출석 트윗 발행**: 매일 오전 10시(서울 시간)에 봇이 출석 트윗 자동 게시
- **답글로 출석**: 유저가 출석 트윗에 답글을 달면 자동으로 재화 지급
- **보상 시스템**:
  - 기본 출석: 50원
  - 연속 출석 보너스:
    - 7일 연속: +20원
    - 14일 연속: +50원
    - 30일 연속: +100원
- **중복 방지**:
  - DB 레벨: UNIQUE(user_id, attendance_post_id) 제약
  - 같은 출석 트윗에 여러 번 답글 달아도 1회만 인정
- **시간 제한**: 출석 트윗 게시 후 23시간 59분까지 유효
- **휴식계 처리**: 휴식 중인 유저도 출석 가능 (활동량 체크와 별개)
- **출석 기록**: 모든 출석은 attendances 테이블에 기록 (reward_amount, streak_days 저장)
- **전역 휴식기간**:
  - calendar_events에서 is_global_vacation=true인 날짜 확인
  - 해당 날짜에는 **출석 트윗 발행 자체를 막음** (cron/celery에서 체크)

### 활동량 체크 (하루 2회, 12시간 간격, Asia/Seoul 타임존 기준)
- 오전 4시(서울 시간): 메인 체크 (벌크)
- 오후 4시(서울 시간): 중간 체크
- 48시간 내 답글 수 확인
- 기준 미달 시 관리자 봇으로 비공개 툿 발송
- role 필터링 (user만 체크, admin/system/story 제외)
- 휴식 중 유저 자동 제외
- **전역 휴식기간**:
  - calendar_events에서 is_global_vacation=true인 날짜 확인
  - 해당 날짜에는 **활동량 체크 자체를 건너뜀** (celery task에서 체크)

## 관리자 웹 기능

### 대시보드
- 전체 통계 (유저 수, 총 재화, 48h 답글)
- 오늘의 알림 (경고 유저, 휴식 중)
- 봇 상태 (정상/오류, 마지막/다음 체크)
- 전체 유저 활동 현황 테이블
  - 🟢 정상 (기준 충족)
  - 🟡 주의 (기준의 70~99%)
  - 🔴 경고 (기준 미달)
  - 💤 휴식 중

### 활동량 관리
- 자동 경고 내역 조회
  - 필터: 오늘/이번주/전체
  - 정렬: 최신순/오래된순
- 수동 경고 발송
  - 유저 선택
  - 메시지 작성 (템플릿 제공)
  - 확인 팝업
  - DM 발송 + warnings 기록
- 휴식 관리
  - 휴식 중 유저 목록
  - 휴식 등록 (시작일, 종료일, 사유)
  - 휴식 해제

### 재화 관리
- 유저별 재화 현황
  - 정렬 (재화 많은순/적은순)
  - 필터 (역할/기숙사)
  - 검색 (유저명)
- 재화 조정
  - 지급/차감
  - 금액 입력
  - 사유 입력 (필수)
  - 확인 팝업
  - transactions + admin_logs 기록

### 상점 관리
- 아이템 목록
  - 카테고리 필터
  - 상태 필터 (판매중/중단)
- 아이템 추가
  - 이름, 설명, 가격
  - 카테고리 (기존 선택 or 신규 생성)
  - 이미지 URL (선택)
  - 판매 상태
- 아이템 수정/삭제
  - 가격 변경
  - 판매 중단/재개
  - 삭제 (확인 팝업)

### 일정/이벤트 관리
- 커뮤니티 일정 등록
  - 제목, 설명, 날짜
  - 이벤트 타입 (일반/공휴일/공지)
  - 전역 휴식기간 설정 체크박스
- 일정 목록
  - 월별 캘린더 보기
  - 주별 리스트 보기
  - 필터 (이벤트 타입별)
- 일정 수정/삭제
  - 기존 일정 편집
  - 삭제 (확인 팝업)
- **전역 휴식기간 관리**
  - 특정 날짜를 전역 휴식기간으로 설정
  - 해당 날짜에는 출석 체크 및 활동량 체크 자동 비활성화
  - 예: 크리스마스, 신년 등 커뮤니티 전체 휴식일 지정

### 시스템 설정
- 활동량 체크 기준
  - 체크 기간 (기본: 48시간)
  - 최소 답글 수 (기본: 20개)
- 재화 지급 설정
  - N개당 M원 (예: 1개당 10원)
- 체크 시간
  - 오전 체크 시간 (기본: 04:00)
  - 오후 체크 시간 (기본: 16:00)
- 출석/활동량 시스템 온/오프
  - 출석 체크 시스템 활성화/비활성화
  - 활동량 체크 시스템 활성화/비활성화
- 모든 설정 DB 저장 (settings 테이블)
- 변경 즉시 적용

### 커뮤니티 건강도 (소셜 분석)
- **대시보드 위젯**
  - 🔴 대화 편중 유저 수
  - 🟡 고립 위험 유저 수
  - 🟢 건강한 유저 수

- **편중 유저 목록**
  - 특정 1명과의 대화 > 30% 유저
  - 대화 상대 수, 최다 대화 상대, 비율 표시
  - 예: "@user1 → @user2와 40% (20/50개)"

- **고립 유저 목록**
  - 48시간 내 대화 상대 < 7명
  - 왕따 위험 유저 조기 발견

- **비활동 유저 목록**
  - 7일 접속률 < 50%
  - 이탈 위험 유저 파악

- **유저 상세 소셜 통계**
  - 대화 상대 분석 (48시간)
  - 접속 패턴 (7일)
  - 경고 이력

- **자동화**
  - 새벽 4시 벌크 분석 (PostgreSQL 쿼리)
  - 편중 유저에게 자동 경고 DM 발송
  - user_stats 테이블에 히스토리 저장

### 콘텐츠 관리
**스토리 / 공지 / 운영진 공지 (3개 탭)**

**스토리**:
- 예약 작성 → 스토리 계정으로 발행
- 공개 범위: public/unlisted
- 예약 목록, 발행 내역

**공지**:
- 예약 작성 → 공지 계정으로 발행
- 공개 범위: public/unlisted
- 예약 목록, 발행 내역

**운영진 공지**:
- 예약 작성 → **관리자 봇으로 비공개 툿 발행**
- 관리자들만 팔로우 → 실시간 알림
- 운영진 전용 정보 공유
- 예약 목록, 발행 내역

### 관리 로그
- 전체 작업 기록 조회
  - 필터 (작업 유형별)
  - 기간 선택
  - 검색 (관리자명/유저명)
- 기록 내용
  - 재화 조정
  - 수동 경고 발송
  - 설정 변경
  - 상점 관리 (아이템 추가/수정/삭제)
  - 휴식 등록/해제
- 엑셀 다운로드

## 역할별 기능

### 일반 유저 (role: user)
- 봇 명령어 사용
- 답글로 재화 획득
- 상점에서 구매
- 활동량 체크 대상

### 관리자 (role: admin)
- 봇 명령어 사용
- 답글로 재화 획득
- 상점에서 구매
- **활동량 체크 제외**
- **관리자 웹 접근**

### 시스템 계정 (role: system)
- 봇 명령어 사용 불가
- 재화 획득 안 됨
- **활동량 체크 제외**
- 예: 봇 계정, 공지 계정

### 스토리 계정 (role: story)
- 봇 명령어 사용 가능
- 답글로 재화 획득
- **활동량 체크 제외**
- 예: 스토리 전용 캐릭터

## 알림 시스템

### 관리자 봇 계정
- 비공개 툿으로 알림 발송
- 관리자들만 팔로우
- 활동량 경고 알림
- 시스템 오류 알림 (선택)

### 대시보드 알림
- 경고 대상 유저 수
- 휴식 중인 유저 수
- 봇 상태 이상 시 표시

## 데이터 기록

### transactions (모든 거래)
- earn_reply: 답글로 획득
- earn_attendance: 출석 체크로 획득
- spend_shop: 상점 구매
- admin_add/remove: 관리자 조정
- 추후: 게임 결과, 양도 등

### warnings (경고 기록)
- auto: 자동 경고
- manual: 수동 경고
- 체크 기간, 요구/실제 답글 수
- DM 발송 여부

### admin_logs (관리 작업)
- adjust_balance: 재화 조정
- send_warning: 수동 경고
- register_vacation: 휴식 등록
- change_settings: 설정 변경
- add/edit/delete_item: 상점 관리
- add/edit/delete_event: 일정 관리

### calendar_events (일정/이벤트)
- 커뮤니티 일정 기록
- 이벤트 타입 (일반/공휴일/공지)
- 전역 휴식기간 표시
- 작성자 기록

## 보안 기능

### OAuth 인증
- 마스토돈 계정으로 로그인
- 비밀번호 직접 입력 없음
- Owner/Admin만 접근 가능

### 권한 체크
- 모든 관리 작업 권한 확인
- 일반 유저 접근 차단
- 세션 만료 자동 처리

### 작업 기록
- 모든 관리 작업 admin_logs 기록
- 누가, 언제, 무엇을 했는지 추적
- 투명한 관리

### 확인 팝업
- 재화 조정 전 확인
- 경고 발송 전 확인
- 아이템 삭제 전 확인
- 실수 방지
