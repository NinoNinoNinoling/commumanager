### 1단계: 서버 접속 & 환경 확인 (10분)
- [ ] SSH 접속: `ssh user@your-server`
- [ ] Python 확인: `python3 --version` (없으면 설치)
- [ ] Git 확인: `git --version` (없으면 설치)
- [ ] 작업 디렉토리 생성: `mkdir -p ~/commumanager && cd ~/commumanager`

### 2단계: 프로젝트 코드 가져오기 (5분)
- [ ] GitHub 클론: `git clone https://github.com/NinoNinoNinoling/commumanager.git .`
- [ ] 브랜치 확인: `git branch -a`
- [ ] 최신 브랜치로 전환: `git checkout claude/merge-latest-to-main-01C2AtQLwUJTWxptzx4zvYW9`

### 3단계: Python 환경 설정 (10분)
- [ ] 가상환경 생성: `python3 -m venv venv`
- [ ] 가상환경 활성화: `source venv/bin/activate`
- [ ] pip 업그레이드: `pip install --upgrade pip`
- [ ] 의존성 설치: `pip install -r requirements.txt`
  - 에러 발생 시 기록 (시스템 패키지 필요할 수 있음)

### 4단계: 환경 변수 설정 (5분)
- [ ] `.env` 파일 생성: `cp .env.example .env`
- [ ] `.env` 편집: `nano .env` 또는 `vi .env`
  ```bash
  SECRET_KEY=랜덤문자열
  MASTODON_CLIENT_ID=마스토돈앱ID
  MASTODON_CLIENT_SECRET=마스토돈앱시크릿
  MASTODON_API_BASE_URL=https://your-mastodon.server
  ```
- [ ] 저장 및 확인

### 5단계: DB 초기화 (5분)
- [ ] DB 생성: `python init_db.py`
- [ ] 테스트 데이터: `python scripts/seed_test_data.py`
- [ ] 확인: `sqlite3 economy.db "SELECT COUNT(*) FROM users;"`
  - 결과: 10명 이상 나와야 함

### 6단계: Admin Web 실행 테스트 (10분)
- [ ] 서버 시작: `cd admin_web && python app.py`
- [ ] 다른 터미널에서 확인: `curl http://localhost:5000`
- [ ] 포트 열기 (방화벽): `sudo ufw allow 5000` (필요시)
- [ ] 외부 접속 테스트: `http://서버IP:5000`
- [ ] Ctrl+C로 서버 종료

### 7단계: OAuth 우회 설정 (테스트용) (5분)
**마스토돈 OAuth 없이 테스트하려면** `admin_web/app.py`에 추가:
```python
@app.route('/debug-login')
def debug_login():
    session['user_id'] = 'test_admin'
    session['username'] = 'TestAdmin'
    return redirect('/dashboard')
```
- [ ] 코드 추가 후 저장
- [ ] 서버 재시작: `python app.py`

- [ ] 사용자 목록 - 10명 표시되는지
- [ ] 사용자 상세 - 거래 내역 확인
- [ ] 에러 나는 페이지 기록

### 9단계: 문제 기록 & 정리 (10분)
- [ ] 발견한 에러 메모
- [ ] 설치 안 된 패키지 목록
