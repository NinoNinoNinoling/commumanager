## 3. Phase 1: 서버 인프라 구축

### 예상 소요: 1~2일

### 3.1 작업 내용

**Day 1: VM 생성 및 기본 설정**
```bash
# 1. VM 인스턴스 생성
- 오라클 춘천 리전
- Ubuntu 22.04 LTS
- 포트 오픈: 22, 80, 443

# 2. 기본 패키지 설치
sudo apt update && sudo apt upgrade -y
sudo apt install -y docker.io docker-compose git nginx certbot

# 3. 방화벽 설정
sudo ufw allow 22/tcp
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw enable
```

**Day 2: 휘핑 에디션 설치**
```bash
# 1. 휘핑 에디션 클론
git clone https://github.com/whippyshou/mastodon.git
cd mastodon

# 2. 환경 설정
cp .env.production.sample .env.production
nano .env.production
# LOCAL_DOMAIN, SMTP 등 설정

# 3. Docker 빌드 및 실행
docker-compose build
docker-compose run --rm web bundle exec rake db:migrate
docker-compose run --rm web bin/tootctl accounts create admin ...
docker-compose up -d

# 4. SSL 설정
sudo certbot --nginx -d yourserver.duckdns.org
```

### 3.2 완료 조건
- [ ] `https://yourserver.duckdns.org` HTTPS 접속 가능
- [ ] 관리자 계정 로그인 성공
- [ ] 일반 유저 가입 테스트 성공
- [ ] PostgreSQL, Redis 정상 작동 확인

### 3.3 트러블슈팅
- Docker 빌드 실패 → RAM 부족 (swap 추가)
- SSL 인증서 실패 → 도메인 DNS 전파 대기
- 마스토돈 접속 안됨 → Nginx 설정 확인
